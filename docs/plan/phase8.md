# Phase 8: MVP 後の将来対応

MVP リリース後に検討する機能拡張。優先度は未定。

## マルチセッション

- [ ] 複数のエージェントセッションを同時に開き、タブで切り替えられるようにする

## 並列エージェント実行

- [ ] 複数のエージェントを同時に走らせて、それぞれの進捗を並べて確認できるようにする

## セッション永続化

- [ ] アプリを再起動してもセッションの会話履歴を復元できるようにする

## ファイル差分ビューア

- [ ] エージェントが書き換えたファイルの変更差分（diff）をアプリ内で確認できるようにする

## 設定パネル

- [ ] kiro-cli の実行パスやアプリのテーマなどをユーザーが設定できる画面を追加する

## readTextFile の部分読み込み対応

- [ ] `ReadTextFileRequest` の `limit`（最大行数）/ `line`（開始行）パラメータに対応し、大きなファイルを分割して読めるようにする
