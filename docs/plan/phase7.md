# Phase 7: ポリッシュとエッジケース

MVP の品質を高めるための仕上げ作業。壊れにくく使いやすいアプリにする。

## エラーハンドリング

- [ ] kiro-cli プロセスが予期せずクラッシュした場合に画面へエラーを表示し、再接続を促す処理を実装する

## キャンセル操作

- [ ] エージェント実行中に停止ボタンまたは Esc キーでキャンセルできるようにする

## スクロール改善

- [ ] ユーザーが自分で上方向にスクロールしている間は自動スクロールを止める処理を実装する（新メッセージが届いても位置を維持する）

## パーミッションダイアログ

- [ ] エージェントから許可を求めてきたとき、自動承認（MVP）ではなくダイアログで承認・拒否できる UI を実装する

## 作業ディレクトリ選択

- [ ] セッション開始時にフォルダ選択ダイアログ（OS ネイティブ）で作業ディレクトリ（`cwd`）を指定できるようにする

## キーボードショートカット整理

- [ ] Enter: プロンプト送信
- [ ] Shift+Enter: 改行
- [ ] Esc: 実行中プロンプトのキャンセル
