## キャンセル操作

- [x] エージェント実行中に停止ボタンでキャンセルできるようにする
  - **AC**: isProcessing が true のとき、送信ボタン（ArrowUp）の代わりに停止ボタン（Square）が表示される
  - **AC**: 停止ボタンをクリックすると `window.kiroductor.session.cancel()` が呼ばれる
  - **AC**: キャンセル後、prompt() が stopReason: "cancelled" で完了し、isProcessing が false に戻る
  - **AC**: キャンセルまでに受信した部分回答はそのまま表示される
